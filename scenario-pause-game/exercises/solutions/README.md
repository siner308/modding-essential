# Exercise Solutions - Pause Game

이 폴더는 scenario-pause-game의 연습문제 해답들을 포함합니다.

## 📋 연습문제 목록

### Exercise 1: 기본 메모리 스캔
**문제**: 게임의 현재 FPS 값을 찾아 읽어오는 프로그램을 작성하세요.

**해답 파일**: `exercise1_fps_reader.cpp`

### Exercise 2: 프로세스 일시정지
**문제**: 특정 프로세스의 모든 스레드를 일시정지하는 함수를 구현하세요.

**해답 파일**: `exercise2_process_pause.cpp`

### Exercise 3: 핫키 시스템
**문제**: F1 키를 눌렀을 때 게임을 일시정지/재개하는 시스템을 만드세요.

**해답 파일**: `exercise3_hotkey_pause.cpp`

### Exercise 4: 안전한 메모리 조작
**문제**: 게임 크래시를 방지하는 안전한 메모리 읽기/쓰기 함수를 작성하세요.

**해답 파일**: `exercise4_safe_memory.cpp`

### Exercise 5: 고급 일시정지
**문제**: 특정 기능만 일시정지하고 UI는 동작하도록 하는 선택적 일시정지를 구현하세요.

**해답 파일**: `exercise5_selective_pause.cpp`

## 📚 학습 목표

각 연습문제를 통해 다음을 학습합니다:

1. **메모리 스캔**: 게임 데이터 구조 이해
2. **프로세스 제어**: Windows API 활용
3. **사용자 인터페이스**: 핫키와 입력 처리
4. **안전성**: 예외 처리와 에러 핸들링
5. **고급 기법**: 선택적 기능 제어

## 🎯 평가 기준

- ✅ **기능성**: 요구사항을 모두 만족하는가?
- ✅ **안전성**: 예외 상황을 적절히 처리하는가?
- ✅ **코드 품질**: 읽기 쉽고 유지보수 가능한가?
- ✅ **성능**: 효율적으로 동작하는가?
- ✅ **사용성**: 사용자가 쉽게 사용할 수 있는가?

## 📝 제출 방법

1. 각 연습문제별로 별도의 `.cpp` 파일 작성
2. 필요한 경우 헤더 파일(`.h`) 추가
3. 빌드 가능한 상태로 제출
4. 간단한 사용법을 주석으로 설명
