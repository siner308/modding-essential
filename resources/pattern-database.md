# 게임 모딩을 위한 패턴 데이터베이스

이 파일에는 다양한 게임에서 발견되는 일반적인 코드 패턴(시그니처)이 포함되어 있으며, 메모리 스캔 및 함수 후킹에 유용합니다.

## 사용 방법:

각 항목은 패턴(바이트 배열)과 마스크(일치시킬 바이트를 나타냄)로 구성됩니다.

- 패턴의 `?`는 와일드카드(상관 없는 바이트)를 나타냅니다.
- 마스크의 `X`는 패턴의 해당 바이트가 일치해야 함을 의미합니다.
- 마스크의 `?`는 해당 바이트가 와일드카드임을 의미합니다.

--- 

### 예제 1: 플레이어 체력 시그니처 (일반)

**설명**: 플레이어 체력을 설정하거나 수정할 수 있는 함수에 대한 시그니처입니다.
**게임**: 일반 (예: 간단한 RPG)

```
Pattern: 8B 45 ?? 89 41 ?? C3
Mask:    X X ? X ? X X
```

--- 

### 예제 2: FPS 카운터 업데이트 (DirectX 11)

**설명**: DirectX 11 게임에서 FPS 카운터를 업데이트하는 일반적인 명령어 시퀀스에 대한 시그니처입니다.
**게임**: 일반 (DirectX 11 기반 게임)

```
Pattern: F3 0F 11 05 ?? ?? ?? ?? 8B 0D ?? ?? ?? ??
Mask:    X X X X ? ? ? ? X X ? ? ? ?
```

--- 

### 예제 3: NOP 슬레드 (코드 케이브 인젝션)

**설명**: NOP(No Operation) 명령어 시퀀스로, 코드 케이브 인젝션의 대상으로 자주 사용됩니다.
**게임**: 일반

```
Pattern: 90 90 90 90 90 90 90 90 90 90
Mask:    X X X X X X X X X X
```

--- 

### 예제 4: 게임 루프 업데이트 (일반)

**설명**: 메인 게임 루프 업데이트 함수에 대한 일반적인 패턴입니다.
**게임**: 일반

```
Pattern: 48 8B C4 48 89 58 20 57 48 83 EC 50
Mask:    X X X X X X X X X X X X
```

--- 

### 예제 5: 무한 탄약 (일반)

**설명**: 탄약을 감소시키는 명령어에 대한 패턴으로, NOPing 또는 수정의 대상이 됩니다.
**게임**: 일반 슈팅 게임

```
Pattern: FF 0C 25 ?? ?? ?? ?? 48 8B C4
Mask:    X X X ? ? ? ? X X X
```
