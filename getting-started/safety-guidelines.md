# 🛡️ 안전한 게임 모딩 가이드라인

게임 모딩은 시스템의 핵심 부분을 다루는 고급 기술입니다. 안전하고 윤리적인 모딩을 위한 필수 가이드라인을 제시합니다.

## 🔒 기본 안전 원칙

### 1. **백업 최우선**

#### **게임 파일 백업**
```bash
# Steam 게임 백업
# Steam → 라이브러리 → 게임 우클릭 → 백업 및 복원

# 수동 백업 (권장)
robocopy "C:\Games\EldenRing" "D:\Backups\EldenRing_original" /E
```

#### **시스템 복원 지점 생성**
```bash
# Windows 시스템 복원 지점 생성
1. 시작 → "복원 지점 만들기" 검색
2. "시스템 복원" 탭 → "만들기"
3. "Modding_Before_[날짜]" 등으로 명명
```

#### **레지스트리 백업**
```bash
# 전체 레지스트리 백업
reg export HKLM backup_HKLM.reg
reg export HKCU backup_HKCU.reg
```

### 2. **격리된 환경 사용**

#### **가상 머신 활용**
```
추천 가상화 소프트웨어:
├── VMware Workstation (유료, 성능 우수)
├── VirtualBox (무료, 기본 기능)
└── Hyper-V (Windows Pro 이상)

VM 설정 권장사항:
├── RAM: 8GB 이상
├── Storage: 100GB 이상
├── Network: NAT 모드 (인터넷 차단)
└── Snapshot 기능 활용
```

#### **샌드박스 환경**
```bash
# Windows Sandbox (Windows 10 Pro 이상)
1. 제어판 → 프로그램 → Windows 기능
2. "Windows Sandbox" 체크
3. 재부팅 후 사용 가능
```

### 3. **네트워크 격리**

#### **인터넷 연결 차단**
```bash
# Windows 방화벽으로 특정 프로그램 차단
1. 고급 보안이 포함된 Windows Defender 방화벽
2. 아웃바운드 규칙 → 새 규칙
3. 프로그램 → 게임 실행 파일 선택
4. 연결 차단 선택
```

#### **Steam 오프라인 모드**
```
Steam → Steam (메뉴) → 오프라인으로 이동
- 게임 다운로드 완료 후 사용
- 온라인 기능 완전 차단
- 안전한 모딩 환경 제공
```

## 🖥️ 시스템 보호

### 1. **권한 최소화**

#### **UAC (사용자 계정 컨트롤) 활용**
```
제어판 → 사용자 계정 → 사용자 계정 컨트롤 설정 변경
├── 추천 레벨: "앱이 컴퓨터를 변경하려는 경우에만 알림"
└── 모든 관리자 권한 요청에 주의 깊게 검토
```

#### **별도 사용자 계정 생성**
```bash
# 모딩 전용 계정 생성
1. 설정 → 계정 → 가족 및 다른 사용자
2. "다른 사용자 추가"
3. 표준 사용자로 생성
4. 모딩 작업은 이 계정에서만 수행
```

### 2. **안티바이러스 설정**

#### **실시간 검사 예외 설정**
```bash
# Windows Defender 예외 추가
1. 설정 → 업데이트 및 보안 → Windows 보안
2. 바이러스 및 위협 방지
3. 바이러스 및 위협 방지 설정 → 설정 관리
4. 제외 추가 또는 제거

예외 대상:
├── 모딩 도구 폴더 (Cheat Engine, x64dbg 등)
├── 게임 폴더 (임시)
└── 작업 중인 프로젝트 폴더
```

⚠️ **주의**: 모딍 작업 완료 후 예외 설정 제거

### 3. **시스템 모니터링**

#### **비정상 활동 감지**
```bash
# 작업 관리자로 모니터링
Ctrl + Shift + Esc
├── 프로세스 탭: CPU/메모리 사용량 확인
├── 성능 탭: 전체 시스템 상태 확인
└── 세부 정보 탭: 의심스러운 프로세스 확인
```

#### **로그 분석**
```bash
# 이벤트 뷰어 확인
1. 시작 → "이벤트 뷰어" 검색
2. Windows 로그 → 시스템
3. 경고 및 오류 이벤트 확인
4. 모딩 작업 전후 비교
```

## 📜 법적 고려사항

### 1. **이용약관 준수**

#### **게임별 정책 확인**
```
확인해야 할 항목들:
├── EULA (End User License Agreement)
├── ToS (Terms of Service)
├── 모딩 정책 (Modding Policy)
└── 커뮤니티 가이드라인

주요 게임사별 정책:
├── Steam: 개인 사용 목적 모딩 일반적으로 허용
├── Epic Games: 게임별 정책 상이
├── FromSoftware: 온라인 기능에 영향 주는 모딩 금지
└── Bethesda: 공식 모딩 도구 제공, 비공식 모딩도 허용적
```

### 2. **저작권 고려사항**

#### **허용 범위**
```
✅ 일반적으로 허용:
├── 개인 학습 목적의 분석
├── 접근성 향상 (시각/청각 장애 지원)
├── 버그 수정 및 성능 개선
└── 개인적 게임 경험 향상

❌ 금지사항:
├── 게임 파일의 재배포
├── DRM 우회
├── 상업적 이용
└── 타인의 온라인 경험 방해
```

### 3. **지역별 법률**

#### **주요 국가별 차이점**
```
미국 (DMCA):
├── Fair Use 조항 존재
├── 교육/연구 목적 일부 허용
└── Reverse Engineering 제한적 허용

유럽 (GDPR):
├── 개인정보 보호 엄격
├── 소프트웨어 역분석 권리 인정
└── 상호 운용성 목적 허용

한국:
├── 저작권법 엄격 적용
├── 개인 사용 목적 예외 조항
└── 영리 목적 사용 엄격 금지
```

## 🤝 윤리적 모딩

### 1. **공정한 게임 환경**

#### **온라인 게임에서의 금지사항**
```
❌ 절대 금지:
├── 치트/핵 사용 (무적, 투시 등)
├── 오토 플레이 (매크로, 봇)
├── 경제 시스템 조작 (아이템 복제 등)
└── 서버 공격 및 방해

✅ 허용 가능한 것들:
├── 시각적 개선 (텍스처, UI)
├── 편의성 향상 (단축키, 정보 표시)
├── 접근성 개선 (색상 조정, 자막)
└── 성능 최적화 (FPS 향상)
```

### 2. **커뮤니티 기여**

#### **건전한 모딩 문화 조성**
```
추천 행동:
├── 지식 공유 (문서화, 튜토리얼)
├── 오픈소스 기여 (코드 공개)
├── 신규 모더 도움 (멘토링)
└── 건설적 피드백 제공

피해야 할 행동:
├── 독점적 지식 보유
├── 악성 코드 배포
├── 커뮤니티 분열 조장
└── 상업적 목적 남용
```

### 3. **개발자 존중**

#### **게임 개발사와의 관계**
```
존중 표현 방법:
├── 원작의 의도 훼손 금지
├── 게임 구매 후 모딩 (불법 복사본 사용 금지)
├── 개발사 정책 준수
└── 건설적 피드백 제공

피해야 할 행동:
├── 게임 크랙 및 배포
├── DLC 우회
├── 개발사 이미지 훼손
└── 공식 업데이트 방해
```

## 🚨 응급 상황 대처

### 1. **시스템 손상 시**

#### **단계별 복구 절차**
```bash
# Level 1: 게임 파일 복구
1. Steam → 게임 우클릭 → 속성
2. 로컬 파일 → 게임 파일 무결성 확인
3. 손상된 파일 자동 복구

# Level 2: 백업에서 복원
1. 백업 폴더에서 원본 파일 복사
2. 게임 폴더의 수정된 파일 교체
3. 게임 재시작 및 정상 작동 확인

# Level 3: 시스템 복원
1. 시작 → "복원" 검색
2. "복원 지점 만들기" 실행
3. "시스템 복원" 탭 → "시스템 복원"
4. 모딩 작업 이전 복원 지점 선택

# Level 4: 재설치
1. 게임 완전 삭제
2. 관련 레지스트리 항목 정리
3. 깨끗한 재설치 수행
```

### 2. **계정 문제 시**

#### **계정 보안 조치**
```bash
# Steam 계정 보안
1. Steam Guard 활성화 확인
2. 비밀번호 변경
3. 로그인 기록 확인
4. 의심스러운 활동 신고

# 게임 계정 정지 시
1. 공식 고객 지원 센터 연락
2. 상황 설명 및 증거 자료 제공
3. 진심어린 사과 및 개선 의지 표명
4. 추가 조치 요청 사항 이행
```

### 3. **법적 문제 시**

#### **대응 절차**
```
1. 즉시 관련 활동 중단
2. 증거 자료 보전 (로그, 스크린샷 등)
3. 법적 전문가 상담
4. 게임사 또는 관련 기관과 소통
5. 필요시 공개 사과 및 재발 방지 약속
```
