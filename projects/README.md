# 프로젝트

이 디렉터리에는 다양한 모딩 관련 프로젝트가 포함되어 있으며, 여러 기술과 기능을 보여줍니다.

## 포함된 프로젝트:

### 1. 게임 오버레이

DirectX 11 후킹을 사용하여 게임 내 오버레이를 만드는 방법을 보여주는 프로젝트입니다. 사용자 정의 UI 요소를 게임 화면에 직접 렌더링할 수 있습니다.

### 2. 모드 프레임워크

게임 모드(DLL)를 로드하고 관리하도록 설계된 기본 모딩 프레임워크입니다. 로깅, 구성 및 이벤트 시스템을 포함하여 모드가 게임 및 서로 상호 작용할 수 있는 공통 API를 제공합니다.

### 3. 간단한 트레이너

기본적인 메모리 읽기/쓰기 기술을 보여주는 간단한 게임 트레이너입니다. 무한 체력 또는 탄약과 같은 치트를 구현하는 데 사용할 수 있습니다.
