# 📚 Reference - 참고 자료

이 폴더에는 게임 모딩 학습에 필요한 다양한 참고 자료와 심화 학습 내용이 포함되어 있습니다.

## 📖 각 자료별 설명

### [어셈블리 퀵 레퍼런스](./assembly-quick-reference.md)

게임 모딩에서 자주 사용되는 x86-64 어셈블리 명령어와 레지스터 사용법을 간략하게 정리한 자료입니다. 메모리 조작, 함수 호출, 조건부 점프 등 핵심 개념을 빠르게 찾아볼 수 있습니다.

### [모딩 접근법 비교](./modding-approaches.md)

다양한 게임 모딩 접근법(콘텐츠 모딩, 스크립트 모딩, 바이너리 모딩 등)의 장단점과 적용 시나리오를 비교 분석한 자료입니다. 어떤 모딩 기법을 선택해야 할지 고민될 때 참고할 수 있습니다.

### [언리얼 엔진 모딩 가이드](./unreal-engine-modding.md)

언리얼 엔진 기반 게임 모딩에 특화된 가이드입니다. PAK 파일 구조, UETools 사용법, 엔진 내부 변수 및 함수 후킹 방법 등 언리얼 엔진 모딩에 필요한 심화 지식을 다룹니다.

## 🚀 심화 학습

이 자료들을 통해 게임 모딩의 깊이 있는 지식을 습득하고, 더 복잡한 모딩 프로젝트에 도전할 수 있습니다.
